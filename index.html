How does Javascript work?

Every brower have own engine in their browser for execute Javascript code. Chrome browser has V8 engine.Javascript is a single threaded language. When JS file is loaded in browser, V8 engine interpret the code line by line from top to bottom. Inside V8 engine, it used JIT(just in line) compiler. Javascript engine parse the code line by line then execute the code.

What does Javascript event loop do?

Javascript's maximum opration are synchronous. But few operations are asynchrounous as like as setTimeOut, setInterval, fetch. The call stack keeps all functions in line for execution.When one function execute and it popped from the stack. Event loop check it that the call stack is empty or not.When it is empty, new functions are added from queue. Otherwise, current function call are processed.

Difference between local and session storage.

Main difference between session storage and local storage is - when user close the browser, all session storage data is cleared.It creates new session and storage for new tab and window. On the other hands, local storage data is not cleared, when the page session ends and it has no expiration date. Both storage keeps data in a key-value pair format.Both storage have the same syntax. Only difference is local storage is accessed with the localStorage variable and session storage is accessed with the sessionStorage variabe.
